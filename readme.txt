WP Theme Exporter is a amazing tool helps you export theme and plugin easily.

== Description ==

If you want to export theme or plugin to a zip with some setting like ignore file extension, ignore folder..., then
WP Theme Exporter is a best choice for you. 


== Installation ==

1. Simply install as a normal WordPress plugin and activate.    
2. Go to Tools/WP Theme Exporter  


== Frequently Asked Questions ==

= Does the plugin lost my files after export?

 No, your file can not be lost.

= I want export a theme, but before export theme i want to push some plugin as zip files to a folder in theme, can plugin do it? =

Yes, you can select some plugin and a theme that you want to do this.
